Frequently Asked Questions

Why do I need an OpenAI API key?

BalanceMeal uses OpenAI's advanced AI models to analyze meals, calculate calories, and generate personalized diet plans. By using your own API key, you have:

  • Complete control over your data
  • Transparent pricing with no hidden fees
  • Direct access to OpenAI's latest models
  • No subscription costs to us

How do I get an OpenAI API key?

  1. Go to platform.openai.com
  2. Create an account or sign in
  3. Navigate to API Keys section
  4. Click "Create new secret key"
  5. Copy the key and paste it into BalanceMeal's OpenAI Settings

How much does OpenAI API usage cost?

Costs depend on which model you choose and how often you use AI features:

  • GPT-3.5 Turbo: ~$0.01-0.02 per meal analysis
  • GPT-4o Mini: ~$0.02-0.03 per meal analysis
  • GPT-4o: ~$0.05-0.10 per meal analysis

Typical monthly usage: $5-15 for regular tracking

Can the app work without an API key?

Yes, but with limited functionality. Without an API key:

  • You can manually enter calorie values
  • Track meals, activities, and weight
  • View weekly statistics

You cannot:

  • Analyze meals from photos or text
  • Generate personalized diet plans
  • Create daily menus

Why aren't my meal photos being analyzed?

Photo analysis requires a vision-capable model. Make sure you're using:

  • GPT-4 Turbo
  • GPT-4o
  • GPT-4o Mini

GPT-3.5 Turbo and standard GPT-4 do not support image analysis.

Is my data private?

Yes, absolutely. All your data is stored locally on your device. The only data sent externally is to OpenAI for meal analysis and plan generation, using your own API key. We never see or store your personal information, meals, or health data.

Do I need to complete the questionnaire?

The questionnaire is optional but highly recommended. Without it, the app works as a basic tracker. With it, you get:

  • Personalized calorie targets based on your goals
  • Custom macro distribution
  • Diet plans that respect your preferences and restrictions
  • Context-aware daily menus

How do diet plans work?

BalanceMeal uses a two-tier system:

  • Diet Plan: A 30-day nutritional framework with your calorie targets and macro distribution
  • Daily Menu: Specific meal suggestions generated daily based on your diet plan and meal history

This ensures consistent nutrition while providing variety.

Can I edit past entries?

Yes! Go to the History tab, tap on any day, and you can edit or delete meals, activities, and weight entries. You can also add new entries to past days.

Why are calorie counts different when I recalculate?

AI models provide estimates based on the information available. Variations can occur due to:

  • Photo quality and angle
  • Description detail level
  • Model interpretation
  • Portion size estimation

For best results, provide both a clear photo and detailed description.

Common Issues

"Invalid API Key" Error

Solutions:

  • Make sure you copied the entire key with no extra spaces
  • Check that your OpenAI account has billing enabled
  • Verify the key hasn't expired or been revoked
  • Try generating a new key from OpenAI platform

App Crashes When Taking Photos

Ensure you've granted camera permissions:

  1. Go to Settings > Privacy & Security > Camera
  2. Find BalanceMeal and enable it
  3. Restart the app

Daily Menu Not Generating

Check that:

  • You have a valid API key entered
  • You've completed the questionnaire
  • You have an active internet connection
  • Your OpenAI account has available credits

Photos Not Saving

Grant photo library access:

  1. Go to Settings > Privacy & Security > Photos
  2. Find BalanceMeal and set to "Add Photos Only" or "Full Access"

System Requirements

  • Internet: Required for AI features
  • Storage: Varies based on number of photos stored
  • OpenAI Account: Required for full functionality
